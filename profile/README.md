# modl.gg
modl.gg is a modern, open-source moderation and support suite built to completely streamline administration for Minecraft servers.

It combines a powerful and slick web interface with AI-driven tools to automate support tickets, manage chat reports, and intelligently handle player punishments, allowing you to stay in control of your server from anywhere.

## Key Features
- Powerful Web Interface: Manage all aspects of your server's moderation from a professional, web-based panel.
- AI-Powered Support: Utilize AI to automatically handle and respond to support tickets and chat reports, saving valuable staff time.
- Dynamic Punishment System: A flexible system built on points and severities.
  - Create unique punishments for Gameplay vs. Social offenses.
  - Configure stacking bans/mutes that execute consecutively.
  - Issue "stat-wiping" bans that reset player data upon expiration.
- Smart Chat Moderation: AI scans reported messages with Minecraft-specific context awareness (e.g., "I'm going to kill you with a fireball" is not a real-world threat) to make suggestions or take automatic action.
- Advanced Player Reporting: Players can submit reports for chat or other rule violations. Staff can view full chat-log snapshots for complete context.
- Custom Forms: Build your own forms for bug reports, support tickets, and staff applications with conditional logic.
- Evidence & File Management: Upload and link evidence (files, YouTube, Imgur) to reports and punishments.
- Full Customization:
  - Manage staff roles with granular permissions for each command and panel page.
  - Customize your panel with a custom logo, favicon, homepage image, and even a custom domain.
- Full plugin locale and message customization.
- Alt Account Linking: Smart system to identify and link alternate accounts.
- Discord Integration: Connect your server's moderation directly to your Discord community.
